Evaluation Metrics Optimization

Align ML metrics to business requirements for the exam.

Precision (Focus on FP minimization)
Use when False Alarms are highly costly (e.g. spam filters, video flagging, customer inconvenience).
Recall / Sensitivity (Focus on FN minimization)
Use when Missing a target is catastrophic (e.g. cancer detection, network outages, financial fraud).
F1-Score (Harmonic Mean)
Use to balance Precision & Recall on highly imbalanced datasets.

Skew vs. Drift Monitoring

Training-Serving Skew

Difference in feature values/schemas between the training dataset and the actual production request data.

Fix: Audit pipeline preprocessing logic to match training.
Prediction Drift

Changes in production input data distribution over time (e.g. user behavior shifts over weeks/months).

Fix: Schedule automatic retraining via Vertex AI Pipelines.
